### 파트분담
1. User: 회원가입, 수정, 탈퇴, 회원 관리 페이지 - 이성준
2. Store: 상품, 구매, 문의, 스토어 관리 페이지 - 김우경, 남윤주(팀장)
3. Interior: 업체, 상담 신청, 인테리어 관리 페이지 - 정종훈
4. Community: 질문하기, 댓글, 신고, 커뮤니티 관리 페이지 - 박빈나
